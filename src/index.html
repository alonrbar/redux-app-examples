redux-app-examples

examples of redux-app together with your favorite client side framework


Example Framwork
1 Counters - a basic example
2 Counters with Identity - using the withId decorator, a "mini ORM" feature
3 Async Dogs - using the sequence decorator, an async pattern that does not require thunks
4 Todo - your familiar todo list example, this time with redux-app (leverages connect and computed)
5 Gladiators - a real world application


Did you know?
Redux-app is framework agnostic.
The exact same code works in Angular, Aurelia and React. Check it out!
